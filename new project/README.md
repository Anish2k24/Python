# 🚗 Vehicle Detection System  

This project is a **Vehicle Detection System** using **Python and OpenCV**. It scans traffic images, detects vehicles, and extracts details using machine learning algorithms.  

## 📌 Features  
✅ Detects multiple vehicles in an image.  
✅ Highlights detected vehicles with bounding boxes.  
✅ Extracts and displays vehicle details on click.  
✅ Uses **OpenCV** and **Machine Learning** for accurate detection.  

## 📂 Project Structure  
```
/vehicle-detection  
│── traffic2.jpg                # Sample traffic image  
│── Vehicle_Detection.csv       # CSV file storing detected vehicle details  
│── vehicle_detection.py        # Python script for vehicle detection  
└── README.md                   # Project documentation  
```
