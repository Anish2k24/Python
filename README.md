
# Python Projects Repository

Welcome to my Python Projects Repository! 👨‍💻🐍

This repository showcases a range of Python projects I have personally developed to explore and demonstrate different programming concepts, problem-solving techniques, and real-world applications.

---

## 📁 Contents

Here are some of the key projects included in this repository:

- **Employee Attendance Tracker.py**  
  A complete attendance tracking system allowing employees to log in/out and generate attendance reports.

- **Library Management System.py**  
  A program to manage library operations including book issuance, returns, and maintaining inventory.

- **Anish Store.py**  
  A virtual shopping store program that calculates total costs and applies conditional discounts.

- **Variables and Data Types.py**  
  Hands-on examples demonstrating Python's core data types and variable usage.

- **Condition Statement.py**  
  Practical examples of `if`, `elif`, and `else` statements to reinforce decision-making logic.

- **Functions.py**  
  User-defined and built-in function examples to explain reusability and modular code practices.

- **Hello World.py**  
  A basic script to get started with Python - the classic first program.

- **Lambda.py**  
  Demonstrates the power of anonymous functions in Python using `lambda` expressions.

- **List Operation.py**  
  Covers basic to advanced list methods including append, insert, remove, sort, and slicing.

- **Loops.py**  
  Demonstrates loop control structures like `for`, `while`, `break`, and `continue`.

- **Prince Inventory Management.py**  
  A product inventory tracker with options to add, remove, and update stock quantities and prices.

- **Tuple & Dictionary.py**  
  Real examples and exercises using immutable tuples and key-value pair dictionaries.

- **Personal Finance Manager.py**  
  Track your monthly income, expenses, and savings. Budget-friendly dashboard for insights and trends.

- **Vehicle Detection System**  
  Uses image processing and machine learning to detect vehicles in traffic images. Includes datasets and model evaluation.

- **Heart Disease Prediction** 🫀 
  A machine learning project using logistic regression to predict heart disease based on health parameters like age, cholesterol, and blood pressure. Includes data preprocessing, model training, and evaluation metrics.

- **House Price Prediction Model** 🏠 (NEW!)  
  A regression-based machine learning model that predicts house prices using features like area, number of bathrooms, garage size, and year built.  
  It includes data preprocessing, feature selection, model training (Linear Regression), evaluation using MSE and R² score, and visualizations for prediction comparison and feature importance.  
  Also includes functionality to predict custom input values and exports results and models using `joblib` and `pickle`.
