Python Projects Repository

This repository contains a collection of Python scripts that demonstrate various programming concepts and applications. Each script is designed to help you learn and apply Python effectively.

Contents:

- **Employee Attendance Tracker.py**: A system for tracking employee attendance, including login, logout, and attendance reports.  
- Library Management System.py**: Program to manage library operations like book issue, return, and inventory.  
- Anish Store.py**: A virtual store program to calculate total shopping costs.  
- Variables and Data Types.py**: Examples of Python variables and data types.  
- Condition Statement.py**: Demonstrates if-elif-else statements with examples.  
- Functions.py**: Examples of Python functions (user-defined and built-in).  
- Hello World.py**: A simple "Hello, World!" program for beginners.  
- Lambda.py**: Examples of lambda functions in Python.  
- List Operation.py**: Operations and methods on Python lists.  
- Loops.py**: Examples of Python loops (`for`, `while`).  
- Prince Inventory Management.py**: Inventory management program with pricing and quantity functionality.  
- Tuple & Dictionary.py**: Operations on tuples and dictionaries.  
- Personal Finance Manager.py**: A tool to help manage personal finances, track income and expenses, set budgets, and analyze savings patterns.
