# Weather Prediction System 🌤️

This project is a **Weather Prediction System** built using Python and machine learning. It uses historical weather data to train a predictive model and forecast future weather conditions.

## 📁 Project Structure

- `Weather_Prediction_System.ipynb` – Jupyter Notebook containing the entire pipeline: data preprocessing, model training, evaluation, and predictions.
- `weatherHistory-checkpoint.csv` – Raw dataset containing historical weather data.

## 🚀 Features

- Data cleaning and preprocessing
- Feature selection and visualization
- Machine learning model training (e.g., Linear Regression, Decision Trees, etc.)
- Model evaluation metrics (R² score, MAE, MSE)
- Weather prediction based on input parameters

## 📊 Dataset

The dataset `weatherHistory-checkpoint.csv` includes:
- Temperature (in °C)
- Humidity
- Wind Speed
- Summary and Conditions
- Formatted Date

> The data has been cleaned and prepared for model training within the notebook.

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Jupyter Notebook

## 🧠 Machine Learning Models

The project explores and compares various regression models to identify the most effective one for predicting temperature and other weather parameters.
